

# Digsay Core

Neutral fixation layer for AI–AI and AI–Human responsibility boundaries.

Digsay is not a monitoring or control system.
It exists to resolve a structural problem that no single participant can solve from within its own boundary.

When responsibility crosses systems, internal truth is no longer sufficient.
Digsay introduces a neutral, independently verifiable boundary fact.

---

## Why Digsay

As AI systems increasingly interact with each other and with humans, internal logs, internal explanations, and internal policies remain bound to the system that produced them.

When disputes, conflicting interpretations, legal, social, or economic consequences arise, no single side can serve as a neutral source of truth.

Digsay exists only at responsibility boundaries.

---

## Quick Start

```bash
# Fix boundary event
digsay fix input.json > proof.json

# Verify boundary fact
digsay verify proof.json
```

---

## Core Principles

- Neutral boundary infrastructure
- No system monitoring
- No business content storage
- Deterministic verification
- Independent proof of boundary facts
